# Simulador de Aplicacao

Uma gem para simular uma aplicação financeira com o CDI, SELIC ou IPCA.

## Instalação

    gem install simulador_aplicacao

## Como Usar

Execute o comando `simulador_aplicacao`  
E depois informe os dados requisitados
